# Sinhala Transliteration Testing Automation

## Student

IT23375314

## Description

This project automates testing of a Sinhala transliteration web application using Playwright and Python.

## Features

* Reads test cases from Excel
* Inputs Singlish text
* Captures Sinhala output
* Compares expected vs actual
* Marks Pass/Fail automatically

## Tools Used

* Python
* Playwright
* openpyxl

## How to Run

pip3 install playwright openpyxl
playwright install
python3 test_automation.py
